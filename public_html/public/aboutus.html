Acerca de Platos Express

Platos express es una plataforma social cuyo objetivo es conectar cocineros amateurs, caseros con la gente comun.
Nuestra mision es que todas aquellas personas cuya pasion sea cocinar encuentre con nosotros un lugar en donde ofrecer sus platos.
Para ello facilitamos una plataforma web o mobile para los usuarios.


Como funciona

Registro

El registro es simple, solo con unos pocos datos personales estas listos, tambien se puede utilizar la opcion de alguna de las redes sociales disponibles como facebook, twitter y google.
Una vez terminado el registro el usuario recibe un mail para activarlo, con esto quedan disponibles todas las funciones del sitio, en el caso de las redes sociales no hace falta activar el usuario pero deben completar email y direccion para activar el usuario.


Comprar un plato

Al ingresar el sitio registrado como usuario, se ven las distintas ofertas de los cocineros, estas estan ordenadas por cercania del usuario,precio y dia de la semana(se ven las ofertas del dia), para ampliar la consulta hay varios filtros como distancia, dia y un buscador por el nombre de plato. El metodo de compra consiste en hacer click sobre el boton comprar, esto agregar el pedido al carrito de compras(arriba a la derecha), al confirmar la compra tanto el vendedor como el comprador reciben una notificacion vai mail,tanto el comprador como el vendedor pueden cancelar la compra, en este caso tambien se notifica via mail. Una vez finalizada la compra desde el sitio web o la aplicacion mobile,el comprador se pone en contacto con el vendedor o viceversa para acordar la entrega, el pago se acuerda entre comprador y vendedor, Platos Express no interviene en el pago del producto adquirido. El ultimo paso para cerrar la venta es la calificacion de ambas partes, uno como vendedor y el otro como comprador.


Como ofrecer un plato

Al ingresar el sitio registrado como usuario, en la parte superior se encuentra el boton Ofrecer Plato, tambien desde el menu Mis Platos/Ofrecer Plato, se cargan los datos del plato, una foto y esta listo para ser vendido.
